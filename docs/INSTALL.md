# Guía de Instalación de cleodocker

Esta guía proporciona instrucciones detalladas para instalar cleodocker en diferentes sistemas operativos.

## Requisitos Previos

-   **macOS**: macOS 11 (Big Sur) o superior. Homebrew instalado.
-   **Linux**: Una distribución moderna como Ubuntu 20.04+, Debian 10+, Fedora 34+, etc. Acceso `root` o `sudo`.
-   **Windows**: Windows 10 (versión 2004) o Windows 11. Virtualización habilitada en la BIOS/UEFI.

**Recursos Recomendados**:
-   2+ núcleos de CPU
-   4+ GB de RAM
-   20+ GB de espacio en disco libre
